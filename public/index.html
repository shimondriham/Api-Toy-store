Toys API Documentation

Regular get routes :

(Without the need for token or Something from the body)

method Request explanation Default
https://toys1234.herokuapp.com/toys View all products ...
https://toys1234.herokuapp.com/toys/?perPage=5 Several products per page 10
https://toys1234.herokuapp.com/toys/?page=2 Page number 1
https://toys1234.herokuapp.com/toys/?sort=price By what to sort the list id
https://toys1234.herokuapp.com/toys/?search=Lego Search by name or info (It is also possible by part of a word) all
https://toys1234.herokuapp.com/toys/cat/for%20Girls Search by category (It is also possible by part of a word) ---
https://toys1234.herokuapp.com/toys/price/?max=50&min=20&page=1 Below or above the required price 0> <10000

Special routes


method Properties (body) token (header) Request explanation
name, email, password Not needed https://toys1234.herokuapp.com/users Add a new user
email, password Not needed https://toys1234.herokuapp.com/toys/login Registered user login
--- Need token https://toys1234.herokuapp.com/toys/userInfo Show products of the same user only
name, price, info, cat, img Need token https://toys1234.herokuapp.com/toys/:id Add a product from the same user
--- Need token https://toys1234.herokuapp.com/toys/:id Add a product from the same user
name, price, info, cat, img Need token https://toys1234.herokuapp.com/toys/:id Product update by the same user
